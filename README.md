# Actor Browser (Lancer Fork)

A fork of the original [Actor Browser](https://github.com/ddbrown30/actor-browser) by ddbrown30, modified for the Lancer system.

## New Features

*   **Lancer Stats**: Added columns for Hull, Agility, Systems, Engineering, Structure, Stress, Heat, and other Lancer-specific metrics.
*   **Context Menu**: Full right-click context menu support (Configure Ownership, View Artwork, Import, Delete, etc.).
*   **Folder Tree View**: Navigation via folder tree structure.

## Installation

1.  Copy the manifest link.
2.  Open Foundry VTT and go to the **Add-on Modules** tab.
3.  Click **Install Module**.
4.  Paste the link into the **Manifest URL** field at the bottom.
5.  Click **Install**.
